USER SECTION:

How to Add an User by an Admin?

To add a new user to the system, follow these steps:

  1. Navigate to the Users of the system.
  2. Click on the Add User button.
  3. Fill in the required details for the new user in the provided form.
  4. Submit the form to add the user to the system.

Once the user is added, they will have the necessary rights to manage, issue, and return components as required.

EMPLOYEE SECTION:

How to Add an Employees to the system by an Admin?

To add a new Employee, follow these steps:

  1. Navigate to the Users of the system.
  2. Click on the Add User button.
  3. Fill in the required details for the new employee in the provided form.
  4. Employees do not have any rights regarding view, manage, issue and return.
  5. Submit the form to add the employee to the system.

    EDIT COMPONENT:

    Once component is created, we can edit the component by clicking on the Edit Button in ‚ÄúAction‚Äù option.

    These modifications can be made by those who have component manage/issue/return rights.

    Edit component consisting of different tabs as follows:

    COMPONENT HISTORY:

    Component History feature enables the admin to track the stock details over time.

    To track the history of component, follow these steps:

    1. Select the component for which you want to view stock details.
    2. Click on the History Button to access the entire stock details of the selected project.
    3. The History can also be viewed by transaction type.

    EXPORT COMPONENT:

    The export csv feature lets you download specific project data as a csv file. This makes it easy to view and work with your project information in a spreadsheet. Additionally, you can check a box to include project histories in the export if needed.

    Steps to export csv file as follows:

    1. Navigate to the Inventory Tab in the Navbar.
    2. Click on the Export Csv Button.
    3. Select the project you want to export.
    4. If you want to include the project history, check the box labeled Include Stock History of Selected Project.
    5. Click on the Export Button to download the file.

    SEARCH COMPONENT:

    The Search Box allows you to find data within the system. Searches can be performed using the following criteria.

  • Manufacturer Part Number
  • Component ID
  • Package
  • Comment
  • Description

    • Steps to do searching are as follows:

    1. Navigate to Search Input Box in the dashboard.
    2. Type your search terms into the input box. You can search by Manufacturer Part Number, Component ID, Package, Comment, or Description.
    3. Click on the Search Button to see the result.

    COMPONENT PAGINATION:

    Pagination helps to split a large amount of data into smaller, more manageable pages as per the results per page in the system.

    The steps are as follows:

    1. Go to the "Results Per Page" section in the dashboard.
    2. You can choose from three options for the number of results per page: 10, 50, or 100.
    3. If you select a limit of 10, the data will be divided into pages with 10 items each. Similarly, selecting 50 or 100 will divide the data into pages with 50 or 100 items respectively.

    BOM (BILL OF MATERIALS):

    A BOM is a detailed list of all components required to create a product.

  • Go to BOM Tab in navbar.
  • BOM is divided into tabs as follows:
    • Consolidate BOM
    • Update BOM Price
    Consolidate BOM:

    Steps to add Consolidate BOM are as follows:

    1. Select the "Consolidate BOM" Tab.
    2. Upload the BOM File: Choose a BOM file that contains the necessary headers.
    3. View File: The selected file will be displayed in a tabular format.
    4. Enter Number of Systems Required: For example, enter "2."
    5. Download File: Click the "Consolidate" button to download the consolidated .xls file.
    Update BOM Price:

    Steps to update BOM Price are as follows:

    1. Select the "Update BOM Price" Tab.
    2. Upload Consolidated BOM File: First, upload the consolidated BOM file you downloaded.
    3. Upload BOM Files: Next, upload the BOM files you selected in the Consolidate BOM step.
    4. View Files: The uploaded files will be displayed in a tabular format.
    5. Enter GST Rate: Input the applicable GST rate.
    6. Download Updated File: Click the "Consolidate" button to download the updated .zip file.

    The Profile Section includes the following subsections:

    • Guide Section
    • Assigned Section
    • History Section
    Guide Section:

    Guide Section provides complete guidance to the user or employee for how to use AIMS.

    To access the Guide Section, steps are as follows:

    1. Login as an employee or user
    2. Navigate to the Profile Tab in Navbar.
    3. Click on the Guide Section.
    4. Click on the Component Inventory Tab to view instructions on how to use AIMS for Component based.
    Assigned Section:

    Assigned Section displays the complete history of components assigned to employees or users within the system.

    To access the Assigned Section, steps are as follows:

    1. Login as an employee or user
    2. Navigate to the Profile Tabin navbar.
    3. Click on the Assigned Section.
    4. A list of the complete assigned history will be displayed.
    History Section:

    The History section provides the complete list of logged-in users or employees' activities.

    To access the History Section, steps are as follows:

    1. Login as an employee or user
    2. Navigate to the Profile Tabin navbar.
    3. Click on the History Section.
    4. The history of the employee or user should be displayed, including details regarding projects, components, stock histories, etc.

    PROFILE SECTION:

    Note: After logging out, you will need to log in again as a user or employee to access the Profile Section.

    The Profile section allows users/employees to view their personal data.

    The Profile Section includes the following subsections:

    • Guide Section
    • Assigned Section
    • History Section
    Guide Section:

    Guide Section provides complete guidance to the user or employee for how to use AIMS.

    To access the Guide Section, steps are as follows:

    1. Login as an employee or user
    2. Navigate to the Profile Tab in Navbar.
    3. Click on the Guide Section.
    4. Click on the IT And Other Inventory Tab to view instructions on how to use AIMS for IT-Inventory based.
    Assigned Section:

    Assigned Section displays the complete history of assets assigned to employees or users within the system.

    To access the Assigned Section, steps are as follows:

    1. Login as an employee or user
    2. Navigate to the Profile Tabin navbar.
    3. Click on the Assigned Section.
    4. A list of the complete assigned history will be displayed.
    History Section:

    The History section provides the complete list of logged-in users or employees' activities.

    To access the History Section, steps are as follows:

    1. Login as an employee or user
    2. Navigate to the Profile Tabin navbar.
    3. Click on the History Section.
    4. The history of the employee or user should be displayed, including details regarding categories, subcategories, transaction types, etc.
